# FinFlow — Концепція та позиціонування платформи

## Суть продукту (Vision)

FinFlow — це комплексна B2B-платформа (CRM/ERP) для автоматизації роботи бухгалтерських аутсорс-агенцій та мікробізнесу. Продукт використовує стратегію "Троянського коня": залучення користувачів через вирішення гострого болю (генерація безкомісійних платіжних діплінків за стандартом НБУ) з подальшим закриттям їх у повноцінну екосистему для управління фінансами, пулом клієнтів (ФОПів) та звітністю.

## Етапи розвитку (Roadmap)

### Фаза 1: Go-to-Market (Лід-магніт)

Утиліта для генерації платіжних посилань (Base64URL) та брендованих QR-кодів. Основна комерційна цінність — Smart Redirect для обходу In-App браузерів (Instagram/Facebook), що усуває тертя при оплаті та замінює еквайринг (економія 2-3%).

### Фаза 2: Core Platform (Утримання)

Мультиакаунтні B2B-дашборди для бухгалтерів. Інструменти для масового управління десятками ФОПів: збереження реквізитів, миттєва генерація рахунків з різними призначеннями платежу, базова аналітика переходів.

### Фаза 3: CRM Екосистема (Масштабування)

Перехід до повноцінного обліку. Інтеграція модулів автоматизації: контроль статусів оплат, нагадування про податкові періоди, генерація актів та API для інтеграції із зовнішніми e-commerce платформами.
