# Зведена таблиця: куди потрапляє юзер після авторизації

| Метод входу | Стан юзера | Redirect після входу |
|------------|------------|---------------------|
| Пароль (Сценарій A) | Існуючий + з паролем | Головна сторінка |
| Magic link (Сценарій B) | Існуючий + без пароля | Профіль (пароль — редагування, решта — перегляд) |
| Magic link (Сценарій C) | Новий юзер | Профіль (повне редагування, ім'я обов'язкове) |
| Forgot password | Існуючий + з паролем | Профіль (пароль — обов'язкове редагування, решта — перегляд) |
| Google OAuth | Новий або існуючий | Головна сторінка |
