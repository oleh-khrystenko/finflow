# Діаграма станів

## Email-авторизація (основний флоу)

```
                        ┌──────────────┐
                        │ Сторінка     │
                        │ авторизації  │
                        │ (email поле) │
                        └──────┬───────┘
                               │
                        ввів email +
                        натиснув "Продовжити"
                               │
                        ┌──────▼───────┐
                        │  Прелоадер   │
                        │  (backend    │
                        │   перевірка) │
                        └──────┬───────┘
                               │
              ┌────────────────┼────────────────┐
              │                │                │
       ┌──────▼──────┐ ┌──────▼──────┐ ┌───────▼──────┐
       │  Сценарій A │ │ Сценарій B  │ │  Сценарій C  │
       │  Є пароль   │ │ Немає       │ │  Новий юзер  │
       │             │ │ пароля      │ │              │
       └──────┬──────┘ └──────┬──────┘ └───────┬──────┘
              │               │                │
       показати поле   "Перевірте пошту" "Перевірте пошту"
       пароля                 │                │
              │               │                │
      ┌───────▼───────┐      │                │
      │ Ввів пароль   │      │                │
      │ [Забув        │      │                │
      │  пароль?]     │      │                │
      └───┬───────┬───┘      │                │
          │       │          │                │
      вірний  невірний       │                │
          │       │          │                │
   ┌──────▼──┐  toast        │                │
   │ Головна │  "Невірний    │                │
   │ сторінка│   пароль"     │                │
   └─────────┘               │                │
                             │                │
                      ┌──────▼────────────────▼──────┐
                      │      Magic link email        │
                      │  (різний текст для B та C)   │
                      └──────────────┬───────────────┘
                                     │
                              клік на посилання
                                     │
                              ┌──────▼───────┐
                              │   Профіль    │
                              │ (protected)  │
                              │ режим        │
                              │ редагування  │
                              └──────────────┘
```

       │
  "Забув пароль?"
       │
  ┌────▼─────────────┐
  │ Magic link email  │

  └────┬─────────────┘
       │
  клік на посилання
       │
  ┌────▼─────────────┐
  │ Профіль           │
  │ Пароль:           │
  │ ОБОВ'ЯЗКОВИЙ      │
  └──────────────────┘
