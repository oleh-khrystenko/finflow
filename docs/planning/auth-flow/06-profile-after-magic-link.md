# Профіль після magic link

Після успішного переходу по magic link юзер автентифікується (отримує сесію) і потрапляє на сторінку профілю (protected route). Стан сторінки залежить від контексту:

## Новий юзер (Сценарій C → magic link)

Профіль **повністю в режимі редагування:**

| Поле | Стан | Обов'язкове | Примітка |
|------|------|-------------|----------|
| Ім'я | Редагування | Так | Потрібне для звернення в листах та на платформі |
| Прізвище | Редагування | Ні | — |
| Аватар | Редагування | Ні | Можливість завантажити |
| Пароль | Редагування | Ні | Рекомендація: "Для безпеки ваших даних ми рекомендуємо встановити пароль" |

```
┌─────────────────────────────────────────┐
│          Заповніть ваш профіль          │
│                                         │
│  Ім'я *                                 │
│  ┌───────────────────────────────────┐  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│                                         │
│  Прізвище                               │
│  ┌───────────────────────────────────┐  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│                                         │
│  Пароль (рекомендовано)                 │
│  ┌───────────────────────────────────┐  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│  Для безпеки ваших даних ми             │
│  рекомендуємо встановити пароль          │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │           Зберегти                │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## Існуючий юзер без пароля (Сценарій B → magic link)

Профіль **частково в режимі редагування:**

| Поле | Стан | Обов'язкове | Примітка |
|------|------|-------------|----------|
| Ім'я | Перегляд (не редагується) | — | Вже заповнене |
| Прізвище | Перегляд (не редагується) | — | Може бути порожнім |
| Аватар | Перегляд (не редагується) | — | — |
| Пароль | Редагування | Ні | Рекомендація: "Для безпеки ваших даних ми рекомендуємо встановити пароль" |

---

## Forgot password (існуючий юзер з паролем → magic link)

Профіль **частково в режимі редагування:**

| Поле | Стан | Обов'язкове | Примітка |
|------|------|-------------|----------|
| Ім'я | Перегляд | — | — |
| Прізвище | Перегляд | — | — |
| Аватар | Перегляд | — | — |
| Пароль | Редагування | Так | Юзер прийшов скинути пароль — поле обов'язкове |
